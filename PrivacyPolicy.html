APIKEY Connect

Privacy Policy

Summary: APIKEY Connect stores all user data (including API keys) locally on your device in encrypted form. We do not collect, transfer, or store any of your data on our servers.

Introduction

This Privacy Policy describes how APIKEY Connect ("we," "our," or "us") handles your information when you use our browser extension. We are committed to protecting your privacy and ensuring the security of your information.

APIKEY Connect is a secure API key management extension that allows users to store their API keys in an encrypted format and use them on authorized websites without exposing sensitive information.

Information We Collect

APIKEY Connect is designed with a "zero-knowledge" approach, meaning we have designed our extension to minimize data collection:

  • API Keys and Credentials: The extension stores your API keys and related information locally on your device in encrypted form.
  • Master Password: Your master password is used to encrypt and decrypt your API keys. We never store, have access to, or transmit your master password.
  • Website Permissions: The extension stores your preferences about which websites can access your API keys.
  • Extension Settings: Configuration preferences you set within the extension are stored locally on your device.

Optional Analytics

The extension includes an optional anonymous usage statistics feature that is disabled by default. If you choose to enable this feature in the advanced settings, we may collect anonymous data about:

  • Extension feature usage (which features are used most frequently)
  • Error occurrences (to help us improve stability)
  • Basic technical information (browser type, operating system)

This anonymous data does not include any personally identifiable information, API keys, or the domains where you use your API keys.

How We Use Information

The information stored by APIKEY Connect is used solely for the purpose of providing the extension's core functionality:

  • Storing and encrypting your API keys
  • Providing your API keys to websites you have authorized
  • Applying your configuration preferences

If you enable optional anonymous usage statistics, we use this data only to:

  • Improve the extension's functionality and user experience
  • Identify and fix bugs
  • Prioritize feature development

Data Storage and Security

We take the security of your data very seriously:

  • Local Storage Only: All your data is stored locally on your device using the browser's secure storage mechanisms. We do not transmit your API keys or master password to our servers or any third-party servers.
  • Encryption: Your API keys are encrypted on your device using AES-GCM encryption with a key derived from your master password using the PBKDF2 algorithm with a high number of iterations.
  • Website Communication: The extension only provides API keys to websites that you have explicitly authorized.

Data Sharing and Transfers

We do not sell, trade, or otherwise transfer your information to outside parties. Since all data is stored locally, we do not have access to your API keys or other sensitive information. The only data transfers that occur are:

  • Between the extension and websites you explicitly authorize to receive your API keys
  • Anonymous usage statistics if you have opted in (contains no personal or sensitive information)

Your Rights and Choices

You have complete control over your data:

  • Access and Delete: You can view all stored API keys through the extension interface, and you can delete individual keys or reset the entire extension at any time.
  • Backup and Export: You can export an encrypted backup of your API keys for safekeeping.
  • Analytics Opt-Out: Usage statistics are disabled by default. You can enable or disable this feature at any time in the extension's Advanced settings.

Children's Privacy

APIKEY Connect is not directed at children under the age of 13, and we do not knowingly collect personal information from children under 13. If you are a parent or guardian and believe we may have collected information from your child, please contact us.

We encourage you to review this Privacy Policy periodically to stay informed about how we are protecting your information.
